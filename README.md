# Migradent Clinic System

Migradent is a dental clinic management backend system built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- MySQL
- JWT Authentication
- Maven

## Features
- User authentication
- Appointment management
- Patient records
- Secure API using JWT

## Setup

1. Clone the repo
2. Configure MySQL in application.properties
3. Run:

mvn spring-boot:run
